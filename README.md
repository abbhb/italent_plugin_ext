# iTalent 考勤工时计算器 —— 浏览器插件

[![Chrome/Edge Manifest V3](https://img.shields.io/badge/Manifest-V3-blue)](https://developer.chrome.com/docs/extensions/mv3/)
[![适用浏览器](https://img.shields.io/badge/浏览器-Chrome%20%7C%20Edge-green)](https://chromewebstore.google.com/)

> 北森 iTalent SaaS 浏览器增强插件（Chrome / Edge），在「我的出勤」页面注入**「📊 计算工时」**按钮，支持对勾选的考勤记录进行工时统计，并以弹窗展示详情。

---

## 功能特性

- **一键统计**：勾选任意考勤记录行，点击按钮即刻计算
- **多维度统计**：
  - 总工时、日均工时
  - 工作日 / 周末分类统计
  - 加班工时（工作日超 8h/天，周末全计）
  - 缺勤合计时长
- **逐日明细**：弹窗内含表格，展示每天的打卡时间、工时、加班量、缺勤量
- **无侵入**：仅注入样式和按钮，不修改页面原有数据，关闭扩展后恢复原样
- **SPA 兼容**：通过 `MutationObserver` 监听 React SPA 路由切换，自动重新注入

## 安装方式

### 方式一：开发者模式加载（推荐，无需上架）

1. 下载或克隆本仓库到本地
   ```bash
   git clone https://github.com/abbhb/italent_plugin_ext.git
   ```

2. 打开浏览器扩展管理页面：
   - **Chrome**：`chrome://extensions/`
   - **Edge**：`edge://extensions/`

3. 开启右上角「**开发者模式**」

4. 点击「**加载已解压的扩展程序**」，选择本仓库根目录（包含 `manifest.json` 的目录）

5. 插件图标出现在工具栏，安装完成 ✅

### 方式二：打包后安装

```bash
# 将仓库根目录打包为 .zip（排除 .git 和 my_kq.html）
zip -r italent_kq_plugin.zip . --exclude "*.git*" --exclude "my_kq.html"
```
然后在扩展管理页拖入 `.crx` 或解压安装。

## 使用方法

1. 打开 iTalent 系统的「**我的出勤**」页面（`https://*.italent.cn/`）
2. 用页面左侧的**复选框**勾选想要统计的考勤记录（支持多选、全选）
3. 点击工具栏中新出现的 **「📊 计算工时」** 按钮

   > 按钮位于现有「补签」按钮旁边

4. 弹窗内查看统计结果：

   | 指标 | 说明 |
   |------|------|
   | 总工时 | 所有选中行工时之和 |
   | 日均工时 | 工作日有效工时 ÷ 有效工作日数 |
   | 工作日加班 | 超出 8h/天 的部分 |
   | 周末加班 | 周末出勤工时（全部计为加班） |
   | 总缺勤 | 缺勤时长汇总 |

5. 点击 **×**、点击遮罩或按 **ESC** 关闭弹窗

## 加班计算规则

| 日期类型 | 规则 |
|---------|------|
| 工作日（周一～周五） | `max(0, 工作时长 - 8)` 小时 |
| 周末（周六、周日） | 工作时长全部计为加班 |

> 标准工时基准默认为 **8 小时/天**，可在 `content_scripts/kq_calculator.js` 顶部修改 `STANDARD_WORK_HOURS` 常量。

## 技术说明

- **Manifest V3**：遵循 Chrome/Edge 最新扩展规范，无后台 Service Worker 需求
- **零依赖**：纯原生 JavaScript（ES2017+），无需 npm/bundler
- **DOM 动态检测**：通过扫描表头行定位「工作时长」等列的 `left` 偏移，而非硬编码，适应列顺序/宽度变化
- **MutationObserver**：监听 React SPA DOM 变化，确保路由切换后按钮持续存在
- **样式隔离**：所有 CSS 类名以 `kq-calc-` 为前缀，避免污染页面原有样式
